### Documentation of the systematic literature review (TXT)
The literature review was conducted following PRISMA 2020 guidelines to identify publications with documented repurposing cases. The documentation includes:
- The definition of topic groups containing synonyms related to the subject, used as keywords
- Topic groups and keywords
- Search strings applied
- Inclusion and exclusion criteria
- Database restrictions (Scopus and Web of Science)
